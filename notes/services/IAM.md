# IAM

## Root Account

- Highest privilege identity.
- Root compromise creates maximum blast radius.
- Use MFA.
- Avoid daily use.

## MFA

- Protects against password compromise.
- Requires multiple factors.

## Access Keys

- Used for CLI, SDKs, APIs, and automation.
- Consist of Access Key ID and Secret Access Key.

## Console Passwords

- Used for AWS Console login.

## Why Engineers Use CLI

- Automation
- Repeatability
- Scripting
- Faster workflows
